Product information

This is an administration dashboard meant to help you manage your portfolio within specific sectors. It provides tools that enable you to actively manage your watchlist, view the top gainers within your sector, view which equities are being purchased by hedge funds, view insider trading reports, analyze financial data, view DCF models of investments, and forecast price action using machine learning algorithms. Some of these features require that you first add some equities to your watchlist using the "Add symbol" button, and then you will be able to use the allocations tab, which tells you the optimal weightings for your equities using a Monte Carlo simulation and historical data. The other tabs have drop downs in order to help you pick certain stocks and sectors to better analyze them.
